## Install

`git clone https://github.com/roalcantara/react-websocket`

### Dependencies

- [git][6]
- [node][7]
- [gitlint][8]
- [pre-commit][9]

## Usage

- Run `npm install` run the tests on the workspace
- Run `npm start` to start the dev server
